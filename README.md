# Exercícios de Python — Nível Básico

Coleção de exercícios introdutórios de Python, cobrindo lógica, estruturas de controle e funções simples.

## Sobre

Este repositório reúne os primeiros exercícios de Python desenvolvidos durante o aprendizado da linguagem. Os exercícios abordam conceitos fundamentais como entrada/saída, condicionais, laços e funções.

## Tecnologias

- Python 3

## Exercícios

| Exercício | Descrição |
|---|---|
| Calculadora simples | Operações matemáticas básicas |
| Gerador de Boas Vindas | Entrada de nome e exibição de mensagem |
| Maior dos números | Comparação entre valores |
| Fatorial | Cálculo do fatorial de um número |
| Chute um número | Jogo de adivinhação interativo |
| Valor hora | Cálculo de pagamento por hora trabalhada |

## Como executar

```bash
python3 nome_do_arquivo.py
```
